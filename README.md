# Snake and Ladder
A Snake and Ladder game built with C# and Windows Forms under the .NET Framework.  
Created as a **school project** to demonstrate practical use of forms, event-driven programming, and fundamental C# concepts. Players roll the dice by clicking a button, and the first to reach tile 100 wins.

# Features
- 2-player gameplay
- Roll dice using a button
- Snakes and ladders mechanics moving players forward or backward
- Turn-based system
- Simple graphical interface using Windows Forms

# Built With
- C#  
- Windows Forms  
- .NET Framework
